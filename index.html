Zipcode to Geolocation (Latitude / Longitude) using GoogleMaps API

Javascript project to return Geo coordinates for a given Zipcode, using Google Maps API.

Stack:
  • Jade (Template)
  • SCSS
  • Javascript
  • GoogleMaps API
  • NPM (JS Modules)
  • Gulp (Minification, Uglify, Assets renaming, Live reload)
